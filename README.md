📌 About This Repository
This repository contains all the tasks completed as part of the Cognifyz Technologies Machine Learning Internship Program. The tasks involve real-world ML problems based on a restaurant dataset, covering regression, classification, recommendation systems, and geographical analysis.
---
✅ Tasks Completed
🔵 Task 1 — Predict Restaurant Ratings
> **Objective:** Build a machine learning regression model to predict the aggregate rating of a restaurant.
Approach:
Handled missing values and encoded categorical variables
Trained 3 models: Linear Regression, Decision Tree, Random Forest
Evaluated using MAE, MSE, RMSE, and R² Score
Analyzed feature importance using Random Forest
Key Results:
Best Model: Random Forest Regressor
Metrics: MAE, RMSE, R² reported in notebook
📁 File: `Task1_Predict_Restaurant_Ratings.ipynb`
---
🟠 Task 3 — Cuisine Classification
> **Objective:** Develop a machine learning model to classify restaurants based on their cuisines.
Approach:
Preprocessed dataset — handled nulls, encoded labels
Trained Logistic Regression and Random Forest Classifier
Evaluated using Accuracy, Precision, Recall, F1-Score
Visualized Confusion Matrix and Feature Importance
Key Results:
Best Model: Random Forest Classifier
Full classification report generated per cuisine type
📁 File: `Task3_Cuisine_Classification.ipynb`
---
🟢 Task 4 — Location-Based Analysis
> **Objective:** Perform geographical analysis of restaurants using latitude & longitude data.
Approach:
Visualized restaurant distribution on an interactive map using Folium
Grouped restaurants by city and analyzed concentration
Calculated average ratings, costs, and cuisines per city
Identified key patterns and insights from location data
Key Results:
Interactive HTML map generated with heatmap overlay
City-wise statistics table produced
Most popular cuisine and highest-rated city identified
📁 File: `Task4_Location_Based_Analysis.ipynb`
